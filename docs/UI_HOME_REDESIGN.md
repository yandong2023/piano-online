# Piano Online Homepage Redesign

## Goal

Move from a utility website to an interactive piano learning product.

## New hierarchy

1. Hero
- Play your first song in 3 minutes
- Start Playing
- Free Play

2. Interactive piano
- Immediate first sound experience
- Minimal controls

3. Popular songs
- Happy Birthday
- Twinkle Twinkle
- Ode to Joy

4. Learning loop
- Learn
- Practice
- Challenge

## Design language

- Dark music stage
- Piano white/black contrast
- Gold accent
- Rounded cards
- Fewer competing actions

## Toolbar

Replace many standalone controls with:

- Current song
- Mode
- Speed
- Sound settings
- More options

## Mobile

Do not shrink desktop layout. Use:

- Song title at top
- Current note hint
- Piano center
- Bottom controls
