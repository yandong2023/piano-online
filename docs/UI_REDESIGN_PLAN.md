# Piano Online UI Redesign Plan

## Goal

Upgrade Piano Online from a tool-style page into an international interactive piano learning product.

## Principles

- First action within 5 seconds: play a note.
- Song learning is the primary journey.
- Chinese and English share the same design system.
- Mobile and desktop have dedicated layouts.

## Design direction

- Dark stage-style piano area.
- Warm accent color for musical feedback.
- Clear hierarchy: play -> learn song -> challenge.
- Avoid SaaS landing page patterns.

## Implementation phases

1. Design tokens and component styles.
2. Home page redesign.
3. Piano toolbar redesign.
4. Song page redesign.
5. Mobile interaction optimization.
