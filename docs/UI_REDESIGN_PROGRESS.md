# Piano Online UI Redesign Progress

## Goal

Transform Piano Online from a utility page into an international interactive piano product.

## Design direction

- Music product instead of SaaS dashboard
- Dark stage + piano focused visual language
- Clear first action: play a song within minutes
- Separate desktop and mobile interaction patterns

## Current phase

1. Design tokens
2. Shared components
3. Homepage redesign
4. Piano toolbar redesign
5. Song page redesign
6. Mobile optimization

## Principles

- One primary action per screen
- Reduce cognitive load
- Song learning is the main growth loop
- English and Chinese share components but not forced layouts
