# Piano Online Homepage Implementation

## Goal

Move from a tool-style landing page to an interactive music product experience.

## New information hierarchy

1. Hero
   - Play your first song in 3 minutes
   - Primary CTA: Start Playing
   - Secondary CTA: Browse Songs

2. Interactive piano
   - Immediate first-note experience
   - Minimal controls

3. Popular songs
   - Happy Birthday
   - Twinkle Twinkle Little Star
   - Ode to Joy
   - Jingle Bells

4. Learning loop
   - Listen
   - Practice
   - Challenge
   - Complete

## Design principles

- Dark music-stage visual language
- One primary action per section
- Mobile-first piano interaction
- English and Chinese share components but not copy
